# CurrencyFromKirillBot
Этот бот предназначен для конвертации валюты.

Начало работы:
1. На главной странице репозитория на GitHub нажмите кнопку Code в правом верхнем углу списка файлов.
2. В открывшемся меню выбрать Download ZIP. После этого весь архив с проектом будет загружен на ПК. 
3. Распаковать архив с помощью любого архиватора и запустить файл bot.py.
4. Через терминал IDE установить и активировать виртуальное  окружение.
   1. ```python -m venv venv```
   2. ```venv\Scripts\activate```
5. Установить зависимости ```pip install -r requirments.txt```
6. Запустить бота.
7. Найти бота в телеграмм "CurrencyBotFromKirill" и написать "/start"
